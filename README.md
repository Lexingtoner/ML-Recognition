# MLRecognition — Детекция лиц на Android

Приложение для Android, демонстрирующее возможности распознавания лиц в реальном времени с использованием **Google ML Kit** и **Jetpack Compose**.

## 🚀 Особенности
- **Детекция в реальном времени**: Использует CameraX для анализа видеопотока.
- **Google ML Kit**: Высокоточное распознавание лиц.
- **Современный стек**: Полностью написано на Kotlin с использованием Jetpack Compose.
- **Управление разрешениями**: Интеграция Accompanist Permissions для обработки доступа к камере.
- **Переключение камер**: Поддержка фронтальной и основной камер.

## 🛠 Технологии
- **Язык**: Kotlin
- **UI**: Jetpack Compose
- **Camera**: CameraX
- **ML**: Google ML Kit (Face Detection)
- **Архитектура**: MVVM (ViewModel, UI State)
- **DI**: (Добавьте, если используете, например, Hilt или Koin)

## 📋 Требования
- Android SDK 29+ (Android 10 и выше)
- Android Studio Ladybug или новее
- Gradle 9.3.1 (обновлено для стабильности)

## 📦 Сборка и запуск
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/MLRecognition.git
   ```
2. Откройте проект в Android Studio.
3. Дождитесь завершения синхронизации Gradle.
4. Запустите приложение на физическом устройстве или эмуляторе с поддержкой камеры.

## 📁 Структура проекта
- `presentation`: UI-компоненты (Compose), ViewModel и состояние экрана.
- `domain`: Логика обработки данных и модели (FaceResult).
- `util`: Вспомогательные функции и расширения.
